Session note

2026-06-30 initialization note

This note captures the setup pass that turned the repository into a fresh docs-first PostgreSQL learning system.

What changed

  • Created a static documentation homepage and curriculum navigation.
  • Generated all 40 session pages from one shared lesson template.
  • Added prompt archive, architecture, references, and review sections.
  • Added a lightweight db/ folder scaffold without jumping ahead into real schema work.

What we understand now

  • The docs are the source of truth for all future work.
  • The course progression is dependency-first and intentionally slow.
  • Every later lesson has a reserved home, even before its SQL files exist.

What we are not allowed to do yet

We are not skipping ahead into application code, ORMs, dashboards, or advanced PostgreSQL features before the early table, query, and modeling lessons have actually been worked through.
