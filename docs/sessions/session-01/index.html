Foundations Session 01

Why databases exist

Explain why a relational database is useful when many people need the same country facts.

Back to session index Next

Learning objective

Explain why a relational database is useful when many people need the same country facts.

What this lesson protects

Without this lesson, later design choices feel like arbitrary commands instead of responses to a real problem.

Concept explanation

A database keeps one source of truth, while files and spreadsheets often spread the same fact across many places.

Underlying database or SQL idea

Learn the problem a database solves before writing any SQL.

PostgreSQL-specific explanation

PostgreSQL gives that source of truth a durable home, SQL language, and rules that protect shared data.
