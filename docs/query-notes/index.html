Query notes

How we explain SQL

Every query example should say what it does, why it exists, and what shape of result it returns.

Reading rule

Start at SELECT, then read FROM, then read filters, joins, grouping, and sorting in the order PostgreSQL applies them conceptually.

Writing rule

Write the smallest correct query first. If the result is wrong, change the query shape before trying to make it prettier.
